NODE.JS - The facts

  • Invented by Ryan Dahl of Joyent, spring 2009
  • A command line tool
  • Based on a couple of C++ libs on top of Googles V8
  • Actually just a thin layer between the C libs and a JavaScript interface to use em!
  • Asynchronous, Non-Blocking I/O (The famous EventLoop - <3 of Node) We will get there in a moment!
  • Designed for the network. Request and Response, cause there is nothing else in the web
  • Heavy use of events! Kinda related to the EventLoop thingy
  • Swiss army knife for network related stuff
  • Can handle thousands of concurrent connections with minimal overhead (CPU/MEMORY) on a single Process
